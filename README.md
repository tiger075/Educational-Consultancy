# Consultancy Registry — GitHub Pages Ready

Upload these three files to your repository root:

- `index.html`
- `styles.css`
- `script.js`

Then publish using GitHub Pages:

1. Open your GitHub repository.
2. Go to **Settings → Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Select branch: `main` or `master`.
5. Select folder: `/root`.
6. Click **Save**.

Your site URL will look like:

`https://YOUR-USERNAME.github.io/YOUR-REPOSITORY/`

Important: GitHub Pages looks for `index.html` by default. If your file is named `consultancy_registry.html`, either rename it to `index.html` or open the full URL ending in `/consultancy_registry.html`.
